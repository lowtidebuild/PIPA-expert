# 면책사항

[English](../en/DISCLAIMER.md) | [한국어](./DISCLAIMER.md)

## 개인 프로젝트

이 저장소는 lowtidebuild의 **독립적인 개인 프로젝트**입니다. 어떠한 로펌, 고용주, 기관, 정부 기관의 후원·보증·제휴와도 무관합니다. 모든 의견, 설계 결정 및 콘텐츠는 전적으로 개인의 것입니다.

## 법률 자문 아님

이 저장소의 어떠한 내용도, 그리고 에이전트가 생성한 법률의견서, 컴플라이언스 평가, 리스크 분석, 개인정보보호법 조문 해석 및 기타 산출물도 법률 자문에 해당하지 않습니다. 이러한 결과물은 **정보 제공 및 업무 보조** 목적으로만 제공됩니다. 자격을 갖춘 법률 전문가의 조언을 대체하는 것으로 신뢰하거나 의존해서는 안 됩니다.

한국 개인정보보호 법령은 복잡하고 빠르게 변화하고 있습니다. 개인정보 보호법, 정보통신망법, 관련 시행령 및 가이드라인은 빈번하게 개정됩니다. 개인정보보호위원회(PIPC)의 집행 관행도 계속 발전하고 있습니다. **귀하의 구체적인 상황에 정통한 법률 전문가만이 신뢰할 수 있는 법률 자문을 제공할 수 있습니다.**

## AI의 한계 — 환각과 오류

이 프로젝트는 대규모 언어 모델(Claude)을 기반으로 합니다. LLM은 실제로 **환각, 사실 오류, 부정확한 법률 분석**을 생성할 수 있습니다. 결과물에는 다음과 같은 문제가 포함될 수 있습니다:

- 개인정보보호법 조문, 시행령, 개보위 가이드라인을 잘못 해석하거나 오인할 수 있음
- 실재하지 않는 조문 번호, 판례 인용, 처분례를 그럴듯하게 생성할 수 있음
- 잘못된 법적 추론을 적용하거나 중요한 예외·적용제외 규정을 누락할 수 있음
- 최근 입법 개정을 반영하지 못할 수 있음
- 서로 다른 법률(개인정보보호법, 정보통신망법, 신용정보법 등)의 조문을 혼동할 수 있음

이 에이전트에는 인용을 KB 원본과 대조하는 **팩트체커 서브에이전트**가 내장되어 있지만, 이 검증이 완벽하지는 않습니다. 어떠한 결과물이든 이를 근거로 행동하기 전에 **반드시 자격을 갖춘 전문가가 직접 검토하고 검증해야 합니다**.

## Knowledge Base 정확성

Knowledge base는 국가법령정보센터 Open API(open.law.go.kr)와 개보위 공식 가이드라인(PDF)에서 수집하여 자동화된 파이프라인(XML 파싱, PDF-to-Markdown 변환)으로 처리하였습니다. 정확성을 위해 최선을 다했으나:

- 자동 파싱 과정에서 포맷 오류가 발생하거나 세밀한 서식(테이블, 각주)이 유실될 수 있음
- Knowledge base는 특정 시점의 스냅샷이며 최신 개정을 포함하지 않을 수 있음
- 개보위 가이드라인의 PDF-to-Markdown 변환 결과가 불완전할 수 있음
- 교차참조 매핑은 알고리즘으로 생성되어 불완전할 수 있음

## 데이터 보안 및 프라이버시

모든 처리는 **로컬 파일시스템 내에서** 이루어지며, 외부 서버나 데이터베이스를 사용하지 않습니다. 다만 에이전트는 LLM API를 호출하므로 **법률 텍스트와 질문이 모델 제공자**(Anthropic)**에게 전송됩니다**.

기밀 문서를 포함한 실제 법률 업무에 이 도구를 사용하려는 경우:

- **최소한**, 무료 티어나 소비자용 플랜이 아닌 **상용 API 키**를 사용하여 표준 상용 데이터 처리 약관이 적용되도록 하세요.
- **이상적으로는**, **Zero Data Retention (ZDR) 계약이 포함된 Enterprise 플랜**을 사용하세요. 단, ZDR이 **완전한 보호를 보장하지는 않습니다**. 데이터는 여전히 제공자 인프라를 경유합니다. 조직 요건에 부합하는지 판단하려면 Anthropic의 [ZDR 및 데이터 처리 문서](https://code.claude.com/docs/en/zero-data-retention)를 직접 확인하세요.
- **반드시**, 기밀정보·특권정보·개인정보를 어떠한 AI 도구로든 처리하기 전에 조직의 **보안·컴플라이언스·IT/DPO 부서**와 사전에 협의하세요.
- 아이러니를 인식하세요: **개인정보보호법 컴플라이언스 도구의 사용 자체가 개인정보보호법을 준수하는지 확인하세요.**

작성자는 다음 사항에 대해 책임을 지지 않습니다:

- API 키, 네트워크 설정 또는 사용자 과실로 인한 데이터 노출
